## Running locally

1. install node and npm
2. install packages within the project

```bash
npm install
```

3. run the DB set up

```bash
node setupdb.js
```

4. convert the csv to db

```bash
python create.py
```

5. run the code

```bash
npm run dev
```

6. open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
